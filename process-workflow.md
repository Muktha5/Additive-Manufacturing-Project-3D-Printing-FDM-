# 🔄 Additive Manufacturing Workflow

## 1. Pre-Processing
- CAD model is created using design software
- Model converted into STL format
- File transferred to printer via software

## 2. Processing (Printing)
- Material is heated and extruded through nozzle
- Layer-by-layer deposition builds the object
- Each layer bonds with the previous layer

## 3. Post-Processing
- Removal of support structures
- Surface finishing
- Cleaning and inspection

---

## ⚠️ Key Factors Affecting Quality
- Layer thickness
- Print speed
- Temperature control
- Material type
